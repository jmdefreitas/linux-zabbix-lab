# Linux Networking Lab

## Goal
Build a virtual Linux lab to practice networking fundamentals:
- Routing
- NAT
- Internal networks
- Linux services

## Environment
- Host RAM: 12 GB
- Hypervisor: VirtualBox
- OS: Lubuntu 24.04.3

## Topology
- Router VM (NAT + Internal)
- Server VM (Internal) lab_net
- Client VM (Internal) lab_net
